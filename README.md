# winform_embed
embed several exe, show it if running, first start if not. 

功能：
加载exe程序，嵌入winform。
如果已经运行，直接显示（需要先嵌入——xml记录是否嵌入 ProcessID="19044" MainWindowHandle="24382854" embedResult="0"），否则先运行起来。
